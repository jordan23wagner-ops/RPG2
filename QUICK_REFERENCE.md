|-----|------|--------|----------|

## 💾 Save System

### Quick Reference
- **Save**: F5 or "Save Game" button
- **Load**: F9 or "Load Game" button
- **Storage**: Browser localStorage
- **Auto-save**: Manual only

### What's Saved
✅ Character stats (HP, Mana, Level, XP)  
✅ All items (inventory + equipped)  
✅ Gold and potions  
✅ Shop upgrades  
✅ Dungeon progress  
✅ Set bonuses  
✅ Current location  

### Save Tips
- Save after boss kills
- Save before risky portal choices
- Save when you get good set items
- Don't rely on auto-save (there isn't one!)

- **Level 1-10**: ~5% per item
- **Level 11-30**: ~10% per item

- **Level 51+**: ~25% per item

## 🏃 Speed Run Strategies

### Fastest Leveling
1. Use Fountains for XP boost
2. Skip tough levels with Portals
3. Farm bosses every 3rd level
4. Prioritize XP-giving enemies

### Set Completion
1. Hunt Set Forges (⚒️)

3. Check every chest
4. Focus on one set at a time

### Gold Farming
1. Avoid Altars (they cost gold)
2. Sell duplicate items

4. Skip low-value loot

## 🔧 Troubleshooting

### Set Bonuses Not Showing
- Verify you have 2+ pieces of the SAME set
- Check the Set Bonuses panel (should auto-appear)
- Try re-equipping items
- Check item descriptions for set name

### POI Not Responding
- Move closer (within 50 pixels)
- Press **F** (not E or Space)
- Verify POI hasn't been used already
- Check you're not in combat

### Save/Load Issues
- Ensure browser allows localStorage
- Try incognito mode if blocked
- Check browser console for errors
- Clear browser cache and re-save

### Boss Not Spawning
- Boss appears every 3rd level (3, 6, 9, etc.)
- Some bosses require minimum player level
- Check you're on a boss level, not cleared it
- Enter dungeon to trigger boss spawn

## 📈 Progression Timeline

### Early Game (Levels 1-15)
- **Focus**: Learn mechanics, collect basic gear
- **Set Goal**: Find first 2-piece bonus
- **Boss Target**: Defeat first 5 bosses
- **POI Priority**: Fountains for XP

### Mid Game (Levels 15-35)
- **Focus**: Complete first full set
- **Set Goal**: 5-piece bonus active
- **Boss Target**: Defeat Lich King (20+), Vampire Lord (25+)
- **POI Priority**: Set Forges, Altars before bosses

### Late Game (Levels 35-50)
- **Focus**: Optimize build, farm best gear
- **Set Goal**: Multiple set options
- **Boss Target**: Fallen Angel (30+), Infernal Titan (40+)
- **POI Priority**: All POIs strategically

### Endgame (Levels 50+)
- **Focus**: Perfect gear, challenge Void Reaver
- **Set Goal**: Mythic Dragonslayer set
- **Boss Target**: Void Reaver, Chaos Wyrm
- **POI Priority**: Forges for perfect rolls

## 🎓 Pro Tips

1. **Save often** - Especially after good RNG (rare drops, boss kills)
2. **One set at a time** - Complete bonuses are stronger than partial ones
3. **Map POIs mentally** - Remember where useful POIs were
4. **Boss preparation** - Use Altar blessing before boss levels
5. **Portal strategy** - Skip when underleveled, take treasure when strong
6. **Inventory management** - Sell low-tier items, keep set pieces
7. **Stat priority** - Speed helps everything (more attacks = more damage/heals)
8. **Combat positioning** - Use POIs as landmarks for kiting
9. **Resource management** - Save potions for bosses, use Shrines for regular healing
10. **Experiment** - Try different sets to find your playstyle
